# [Description](https://leetcode.cn/problems/online-stock-span/description/)

## Code

形式很新颖，从左到右动态加载价格，可以记录下标与价格，用单调栈维护之前的价格，入栈的时候更新结果

```cpp
class StockSpanner {
public:
    StockSpanner() {
        // 题目限制了从左往右加载数组
        idx = 0;
    }
    
    int next(int price) {
        while (!stk.empty() && price >= stk.top().second) {
            stk.pop();
        }
        int result = stk.empty() ? idx + 1 : idx - stk.top().first;
        stk.push({idx++, price});
        return result;
    }
private:
    std::stack<std::pair<int, int>> stk;
    int idx;
};

/**
 * Your StockSpanner object will be instantiated and called as such:
 * StockSpanner* obj = new StockSpanner();
 * int param_1 = obj->next(price);
 */
```
