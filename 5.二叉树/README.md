# 一些注意事项

## 1. Lambda函数

二叉树相关题目总是涉及到许多递归解法，但是要注意少用lambda函数进行深度递归，这一点在[1373题](./1373.%20二叉搜索子树的最大键值和.md)体现的尤为明显，其中有AI给出的详细解释与使用建议，为了方便在这里贴一下：

- **递归算法不要用 std::function 包裹 lambda**，尤其是深度递归时。
- 推荐用普通成员函数递归，或者用静态函数+参数传递。
- lambda 适合轻量、非递归、无捕获的场景。
